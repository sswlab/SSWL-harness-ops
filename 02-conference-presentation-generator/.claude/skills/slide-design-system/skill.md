# Slide-Design-System — 학술 발표 디자인 시스템

## 개요

SSWL 학술 발표 슬라이드의 일관된 디자인을 위한 색상, 폰트, 레이아웃, 그리드 규칙을 정의한다. slide-composer(레이아웃 지정)와 pptx-engineer(코드 구현)가 이 스킬을 참조한다.

---

## 슬라이드 규격

| 항목 | 값 |
|---|---|
| 비율 | 16:9 |
| 크기 | 13.333" × 7.5" (1280 × 720pt) |
| 여백 | 상하좌우 0.5" |
| 콘텐츠 영역 | 12.333" × 6.5" |
| 제목 영역 | 상단 0.5" ~ 1.3" (높이 0.8") |
| 본문 영역 | 1.5" ~ 7.0" |
| 푸터 영역 | 하단 0.3" (슬라이드 번호, 학회명) |

## 레이아웃 패턴 (10종)

### 1. Title — 타이틀 슬라이드
- 배경: NAVY
- 중앙: 제목 (44pt Bold White) + 부제 (22pt Light_BG) + 저자·소속
- 하단: 학회명, 날짜
- 우하단: 로고 (있으면)

### 2. Section — 섹션 구분
- 배경: BLUE
- 중앙: 섹션 제목 (36pt Bold White)
- 용도: "Introduction", "Data & Methods", "Results" 등 전환

### 3. Text Only — 텍스트 중심
- 배경: WHITE
- 상단: 제목 (28pt NAVY)
- 본문: 불릿 3~5개 (20pt BLACK)
- 여백: 좌 0.8", 우 0.8"

### 4. Text + Image — 텍스트+이미지 분할
- 좌측 (40~50%): 제목 + 불릿
- 우측 (50~60%): Figure
- Figure 캡션: 하단 12pt GRAY

### 5. Image Full — Figure 중심
- 상단: 제목 (28pt)
- 중앙: Figure (슬라이드 70%+ 면적)
- 하단: 캡션
- 용도: 핵심 결과 Figure

### 6. Comparison — 비교
- 좌우 2분할 또는 상하 2분할
- 동일한 크기의 두 Figure/텍스트 블록
- 용도: Baseline vs Experiment, Before vs After

### 7. Key Number — 핵심 수치
- 중앙: 큰 숫자 (72~96pt NAVY Bold)
- 하단: 한 줄 설명 (20pt)
- 용도: 정확도, 개선율 등 임팩트 있는 수치

### 8. Table — 데이터 테이블
- 상단: 제목
- 중앙: 테이블 (최대 7행 × 5열)
- 헤더: NAVY 배경 + WHITE 텍스트
- 본문: 교대행 (WHITE / TABLE_ALT)

### 9. Timeline — 타임라인/프로세스
- 가로 화살표 + 단계별 아이콘/텍스트
- 용도: 분석 파이프라인, 데이터 처리 흐름

### 10. Closing — 마무리
- 배경: NAVY
- 중앙: "Thank you" / "Questions?"
- 하단: 연락처 (이메일, ORCID)

## Figure 삽입 규칙

| 규칙 | 상세 |
|---|---|
| 원본 비율 유지 | 가로/세로 비율 변형 금지 |
| 최소 해상도 | DPI 150 이상 (인쇄: 300) |
| 배경 | 흰색 배경 Figure 권장 (슬라이드와 자연스러운 통합) |
| 캡션 | Figure 하단에 12pt GRAY로 짧은 설명 |
| 테두리 | 없음 (깔끔한 외관) |
| 최대 크기 | 슬라이드 면적의 80% 이내 |
